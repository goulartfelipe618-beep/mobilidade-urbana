# Transporte.PRO Admin

Painel administrativo operacional.

## Rodar

```bash
cd apps/admin-web
npm install
npm run dev
```

Porta **5175** com proxy `/api` para `http://localhost:3000`.

## Escopo

- cockpit executivo e operacional;
- monitoramento de corridas, motoristas, passageiros, pricing, pagamentos, risco e realtime;
- checklists baseados no `guia.txt`;
- health checks do backend e endpoints principais.
